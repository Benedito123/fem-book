Appendix: Useful formulas¶

Finite difference operator notation¶

All the finite differences here, and their corresponding operator notation. take place on a time mesh for a function of time. The same formulas apply in space too (just replace \(t\) by a spatial coordinate and add spatial coordinates in \(u\)).

[kam 51: trenger å sette navn ved de forskjellge formlene. Noen steder er det \("="\) og andre steder er det $”approx”$]

\[\tag{488} u'(t_n) \approx \lbrack D_tu\rbrack^n = \frac{u^{n+\frac{1}{2}} - u^{n-\frac{1}{2}}}{\Delta t}\]
\[\tag{489} u'(t_n) \approx \lbrack D_{2t}u\rbrack^n = \frac{u^{n+1} - u^{n-1}}{2\Delta t}\]
\[\tag{490} u'(t_n) = \lbrack D_t^-u\rbrack^n = \frac{u^{n} - u^{n-1}}{\Delta t}\]
\[\tag{491} u'(t_n) \approx \lbrack D_t^+u\rbrack^n = \frac{u^{n+1} - u^{n}}{\Delta t}\]
\[\tag{492} u'(t_{n+\theta}) = \lbrack \bar D_tu\rbrack^{n+\theta} = \frac{u^{n+1} - u^{n}}{\Delta t}\]
\[\tag{493} u'(t_n) \approx \lbrack D_t^{2-}u\rbrack^n = \frac{3u^{n} - 4u^{n-1} + u^{n-2}}{2\Delta t}\]
\[\tag{494} u''(t_n) \approx \lbrack D_tD_t u\rbrack^n = \frac{u^{n+1} - 2u^{n} + u^{n-1}}{\Delta t^2}\]
\[\tag{495} u(t_{n+\frac{1}{2}}) \approx \lbrack \overline{u}^{t}\rbrack^{n+\frac{1}{2}} = \frac{1}{2}(u^{n+1} + u^n)\]
\[\tag{496} u(t_{n+\frac{1}{2}})^2 \approx \lbrack \overline{u^2}^{t,g}\rbrack^{n+\frac{1}{2}} = u^{n+1}u^n\]
\[\tag{497} u(t_{n+\frac{1}{2}}) \approx \lbrack \overline{u}^{t,h}\rbrack^{n+\frac{1}{2}} = \frac{2}{\frac{1}{u^{n+1}} + \frac{1}{u^n}}\]
\[\tag{498} u(t_{n+\theta}) \approx \lbrack \overline{u}^{t,\theta}\rbrack^{n+\theta} = \theta u^{n+1} + (1-\theta)u^n ,\]
\[\tag{499} \qquad t_{n+\theta}=\theta t_{n+1} + (1-\theta)t_{n-1}\]

Truncation errors of finite difference approximations¶

\[{u_{\small\mbox{e}}}'(t_n) = [D_t{u_{\small\mbox{e}}}]^n + R^n = \frac{{u_{\small\mbox{e}}}^{n+\frac{1}{2}} - {u_{\small\mbox{e}}}^{n-\frac{1}{2}}}{\Delta t} +R^n\nonumber,\]
\[\tag{500} R^n = -\frac{1}{24}{u_{\small\mbox{e}}}'''(t_n)\Delta t^2 + {\cal O}(\Delta t^4)\]
\[{u_{\small\mbox{e}}}'(t_n) = [D_{2t}{u_{\small\mbox{e}}}]^n +R^n = \frac{{u_{\small\mbox{e}}}^{n+1} - {u_{\small\mbox{e}}}^{n-1}}{2\Delta t} + R^n\nonumber,\]
\[\tag{501} R^n = -\frac{1}{6}{u_{\small\mbox{e}}}'''(t_n)\Delta t^2 + {\cal O}(\Delta t^4)\]
\[{u_{\small\mbox{e}}}'(t_n) = [D_t^-{u_{\small\mbox{e}}}]^n +R^n = \frac{{u_{\small\mbox{e}}}^{n} - {u_{\small\mbox{e}}}^{n-1}}{\Delta t} +R^n\nonumber,\]
\[\tag{502} R^n = -\frac{1}{2}{u_{\small\mbox{e}}}''(t_n)\Delta t + {\cal O}(\Delta t^2)\]
\[{u_{\small\mbox{e}}}'(t_n) = [D_t^+{u_{\small\mbox{e}}}]^n +R^n = \frac{{u_{\small\mbox{e}}}^{n+1} - {u_{\small\mbox{e}}}^{n}}{\Delta t} +R^n\nonumber,\]
\[\tag{503} R^n = -\frac{1}{2}{u_{\small\mbox{e}}}''(t_n)\Delta t + {\cal O}(\Delta t^2)\]
\[{u_{\small\mbox{e}}}'(t_{n+\theta}) = [\bar D_t{u_{\small\mbox{e}}}]^{n+\theta} +R^{n+\theta} = \frac{{u_{\small\mbox{e}}}^{n+1} - {u_{\small\mbox{e}}}^{n}}{\Delta t} +R^{n+\theta}\nonumber,\]
\[R^{n+\theta} = -\frac{1}{2}(1-2\theta){u_{\small\mbox{e}}}''(t_{n+\theta})\Delta t + \frac{1}{6}((1 - \theta)^3 - \theta^3){u_{\small\mbox{e}}}'''(t_{n+\theta})\Delta t^2 + \nonumber\]
\[\tag{504} \quad {\cal O}(\Delta t^3)\]
\[{u_{\small\mbox{e}}}'(t_n) = [D_t^{2-}{u_{\small\mbox{e}}}]^n +R^n = \frac{3{u_{\small\mbox{e}}}^{n} - 4{u_{\small\mbox{e}}}^{n-1} + {u_{\small\mbox{e}}}^{n-2}}{2\Delta t} +R^n\nonumber,\]
\[\tag{505} R^n = \frac{1}{3}{u_{\small\mbox{e}}}'''(t_n)\Delta t^2 + {\cal O}(\Delta t^3)\]
\[{u_{\small\mbox{e}}}''(t_n) = [D_tD_t {u_{\small\mbox{e}}}]^n +R^n = \frac{{u_{\small\mbox{e}}}^{n+1} - 2{u_{\small\mbox{e}}}^{n} + {u_{\small\mbox{e}}}^{n-1}}{\Delta t^2} +R^n\nonumber,\]
\[\tag{506} R^n = -\frac{1}{12}{u_{\small\mbox{e}}}''''(t_n)\Delta t^2 + {\cal O}(\Delta t^4)\]
\[{u_{\small\mbox{e}}}(t_{n+\theta}) = [\overline{{u_{\small\mbox{e}}}}^{t,\theta}]^{n+\theta} +R^{n+\theta} = \theta {u_{\small\mbox{e}}}^{n+1} + (1-\theta){u_{\small\mbox{e}}}^n +R^{n+\theta},\nonumber\]
\[\tag{507} R^{n+\theta} = -\frac{1}{2}{u_{\small\mbox{e}}}''(t_{n+\theta})\Delta t^2\theta (1-\theta) + {\cal O}(\Delta t^3) {\thinspace .}\]

Finite differences of exponential functions¶

Complex exponentials¶

Let \(u^n = \exp{(i\omega n\Delta t)} = e^{i\omega t_n}\).

\[\tag{508} [D_tD_t u]^n = u^n \frac{2}{\Delta t}(\cos \omega\Delta t - 1) = -\frac{4}{\Delta t}\sin^2\left(\frac{\omega\Delta t}{2}\right),\]
\[\tag{509} [D_t^+ u]^n = u^n\frac{1}{\Delta t}(\exp{(i\omega\Delta t)} - 1),\]
\[\tag{510} [D_t^- u]^n = u^n\frac{1}{\Delta t}(1 - \exp{(-i\omega\Delta t)}),\]
\[\tag{511} [D_t u]^n = u^n\frac{2}{\Delta t}i\sin{\left(\frac{\omega\Delta t}{2}\right)},\]
\[ \begin{align}\begin{aligned}\tag{512} [D_{2t} u]^n = u^n\frac{1}{\Delta t}i\sin{(\omega\Delta t)}\\ {\thinspace .}\end{aligned}\end{align} \]

Real exponentials¶

Let \(u^n = \exp{(\omega n\Delta t)} = e^{\omega t_n}\).

\[\tag{513} [D_tD_t u]^n = u^n \frac{2}{\Delta t}(\cos \omega\Delta t - 1) = -\frac{4}{\Delta t}\sin^2\left(\frac{\omega\Delta t}{2}\right),\]
\[\tag{514} [D_t^+ u]^n = u^n\frac{1}{\Delta t}(\exp{(i\omega\Delta t)} - 1),\]
\[\tag{515} [D_t^- u]^n = u^n\frac{1}{\Delta t}(1 - \exp{(-i\omega\Delta t)}),\]
\[\tag{516} [D_t u]^n = u^n\frac{2}{\Delta t}i\sin{\left(\frac{\omega\Delta t}{2}\right)},\]
\[ \begin{align}\begin{aligned}\tag{517} [D_{2t} u]^n = u^n\frac{1}{\Delta t}i\sin{(\omega\Delta t)}\\ {\thinspace .}\end{aligned}\end{align} \]

Finite differences of \(t^n\)¶

The following results are useful when checking if a polynomial term in a solution fulfills the discrete equation for the numerical method.

\[\tag{518} \lbrack D_t^+ t\rbrack^n = 1,\]
\[\tag{519} \lbrack D_t^- t\rbrack^n = 1,\]
\[\tag{520} \lbrack D_t t\rbrack^n = 1,\]
\[\tag{521} \lbrack D_{2t} t\rbrack^n = 1,\]
\[ \begin{align}\begin{aligned}\tag{522} \lbrack D_{t}D_t t\rbrack^n = 0\\ {\thinspace .}\end{aligned}\end{align} \]

The next formulas concern the action of difference operators on a \(t^2\) term.

\[\tag{523} \lbrack D_t^+ t^2\rbrack^n = (2n+1)\Delta t,\]
\[\tag{524} \lbrack D_t^- t^2\rbrack^n = (2n-1)\Delta t,\]
\[\tag{525} \lbrack D_t t^2\rbrack^n = 2n\Delta t,\]
\[\tag{526} \lbrack D_{2t} t^2\rbrack^n = 2n\Delta t,\]
\[\tag{527} \lbrack D_{t}D_t t^2\rbrack^n = 2,\]

Finally, we present formulas for a \(t^3\) term: These must be controlled against lib.py. Use \(t_n\) instead of \(n\Delta t\)??

\[\tag{528} \lbrack D_t^+ t^3\rbrack^n = 3(n\Delta t)^2 + 3n\Delta t^2 + \Delta t^2,\]
\[\tag{529} \lbrack D_t^- t^3\rbrack^n = 3(n\Delta t)^2 - 3n\Delta t^2 + \Delta t^2,\]
\[\tag{530} \lbrack D_t t^3\rbrack^n = 3(n\Delta t)^2 + \frac{1}{4}\Delta t^2,\]
\[\tag{531} \lbrack D_{2t} t^3\rbrack^n = 3(n\Delta t)^2 + \Delta t^2,\]
\[\tag{532} \lbrack D_{t}D_t t^3\rbrack^n = 6n\Delta t,\]

Application of finite difference operators to polynomials and exponential functions, resulting in the formulas above, can easily be computed by some sympy code:

from sympy import *
t, dt, n, w = symbols('t dt n w', real=True)

# Finite difference operators

def D_t_forward(u):
    return (u(t + dt) - u(t))/dt

def D_t_backward(u):
    return (u(t) - u(t-dt))/dt

def D_t_centered(u):
    return (u(t + dt/2) - u(t-dt/2))/dt

def D_2t_centered(u):
    return (u(t + dt) - u(t-dt))/(2*dt)

def D_t_D_t(u):
    return (u(t + dt) - 2*u(t) + u(t-dt))/(dt**2)


op_list = [D_t_forward, D_t_backward,
           D_t_centered, D_2t_centered, D_t_D_t]

def ft1(t):
    return t

def ft2(t):
    return t**2

def ft3(t):
    return t**3

def f_expiwt(t):
    return exp(I*w*t)

def f_expwt(t):
    return exp(w*t)

func_list = [ft1, ft2, ft3, f_expiwt, f_expwt]

To see the results, one can now make a simple loop like

for func in func_list:
    for op in op_list:
        f = func
        e = op(f)
        e = simplify(expand(e))
        print e
        if func in [f_expiwt, f_expwt]:
            e = e/f(t)
        e = e.subs(t, n*dt)
        print expand(e)
        print factor(simplify(expand(e)))
